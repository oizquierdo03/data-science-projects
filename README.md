# Data Science Projects

This repository contains a collection of data science and machine learning projects developed during my coursework. The projects cover data visualization, predictive modeling, and classification techniques applied to real-world datasets.

---

## 📊 Projects

### 1. Crime Data Visualization
Exploratory data analysis and visualization of crime data in Montreal.

- Data cleaning and preprocessing  
- Statistical visualization of crime patterns  
- Geographic visualization using maps  

---

### 2. Laptop Price Prediction
Regression models to predict laptop prices based on technical specifications.

- Feature engineering and preprocessing  
- Model training using regression techniques  
- Evaluation of model performance  

---

### 3. Fraud Detection
Machine learning models to detect fraudulent transactions in financial datasets.

- Data preprocessing and class imbalance handling  
- Classification models (e.g., Random Forest)  
- Evaluation using precision, recall, and other metrics
