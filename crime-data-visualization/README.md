# Crime Data Visualization

This project explores and visualizes crime data in Montreal using statistical and geographic tools.

---

## 📊 Dataset

The dataset used in this project can be found here:

[https://github.com/marcoaaceves/DataScience/tree/main/Crime%20Data%20Analysis]

---

## ⚙️ Description

- Data cleaning and preprocessing  
- Visualization of crime patterns  
- Geographic mapping of incidents
