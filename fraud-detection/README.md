# Fraud Detection

This project focuses on detecting fraudulent transactions using machine learning classification models.

---

## 📊 Dataset

The dataset used in this project can be found here:

[https://github.com/Fraud-Detection-Handbook/simulated-data-raw/tree/main/data]

---

## ⚙️ Description

- Preprocessing of transaction data  
- Handling class imbalance  
- Training classification models (e.g., Random Forest)  
- Evaluation using precision and recall
